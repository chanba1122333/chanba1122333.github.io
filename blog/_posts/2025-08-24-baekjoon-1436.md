## 백준 1436번: 영화감독 숌

[문제 링크](https://www.acmicpc.net/problem/1436)


### 코드

```cpp
# 백준 1436번: 영화감독 숌 

#include <bits/stdc++.h>

using namespace std;
typedef long long ll;

int main(void)
{
    ios::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);
    ll n;
    cin >> n;
    ll count = 0;
    ll num = 666;
    while (1)
    {
        string s = to_string(num);
        if (s.find("666") != string::npos)
        {
            count++;
        }
        if (count == n)
        {
            cout << num << '\n';
            break;
        }

        num++;
    }
    return 0;
}
```
